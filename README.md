Project Overview

This project focuses on analyzing hospitality business performance using an interactive Excel dashboard. The goal is to track revenue, bookings, occupancy, and cancellations to support data-driven decision-making for hotel operations.

Business Problem

Hospitality businesses need clear visibility into booking trends, occupancy rates, and revenue performance. Raw booking data is often unstructured, making it difficult to analyze performance across hotels, cities, and time periods.

Data Preparation

  1.Cleaned and transformed raw hospitality data using Power Query
  2.Removed inconsistencies and standardized date and category fields
  3.Prepared structured datasets for accurate analysis and reporting

Dashboard Features

KPI metrics for Revenue, Bookings, Occupancy Rate, Capacity, and Cancellation Rate
City-wise and property-wise performance analysis
Weekday vs weekend revenue comparison
Monthly and weekly trend analysis using Pivot Charts
Interactive slicers for dynamic filtering

Tools Used

Microsoft Excel
Power Query
Pivot Tables
Pivot Charts

Key Insights

  1.Identified high-performing hotels and cities based on revenue and occupancy
  2.Observed seasonal and weekday/weekend booking trends
  3.Analyzed cancellation patterns impacting overall revenue

Conclusion

This Excel dashboard provides a structured and interactive view of hospitality performance, enabling stakeholders to monitor KPIs, identify trends, and make informed business decisions.
